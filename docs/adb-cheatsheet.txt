#Adb imp commands::
:adb devices  (device visibility command )
 if it shows - device is attachd then path of adb is set perfectly but if shows no recognised then set adb path correctly 

: adb install appname.apk    
..used to install apk on your device

:adb install -r appname.apk   
 used to reinstall the app 

:adb uninstall app_pkg_name 
used to unintsall the  app from device

:adb shell pm list packages 
 display all packages of  apps in your device.if you didnt find specific app package then do ctrl+f and 
write  com.appname 

:adb shell pm clear (pkg name of specifc app) ....used to clear app data 

:adb shell am force-stop pkg name ...used to stop app from background

:adb shell monkey -p pkg_name -c android.intent.category.LAUNCHER 1
used to open app same like user open the app 


:adb logcat ....- start logs and ctrl+c for stop logcats 

:adb logcat -c ... to clear old logs .. purane logs ko delete kr deta ha taki ham new se start kr ske 

:adb logcat *:E..to filter the error 
:adb logcat -d> file.txt (here is the file where your logs  saved)


:adb logcat | findstr pkg_name 
used for filter the logs
